## Installation

    $ cd ~
    $ git clone git://github.com/raglub/vimrc .vim
    $ git submodule init
    $ git submodule update
    $ vim .vimrc

To file ".vimrc" add a line

    so ~/.vim/vimrc
## ShortCuts
  <F4> - Open/Close Gundo
  <F5> - Open/Close NerdTree
