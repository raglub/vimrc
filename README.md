## Installation

```bash
    cd ~
    git clone https://github.com/raglub/vimrc .vim
    cd ~/.vim
    git submodule init
    git submodule update
    vim ~/.vimrc
```

To file ".vimrc" add a line

```
    so ~/.vim/vimrc
```

## ShortCuts

    <F4> - Open/Close Gundo
    <F5> - Open/Close NerdTree
    <F6> - Show differences to change the file
    <c-F10> - Open CtrlP
